# AnonUpload
A PowerShell Module to Upload to [anonfiles.com](https://anonfiles.com/)


### Script Example
```ps
AnonUpload -File .\temp.txt
```

### How to install:
```powershell
Install-Module -Name AnonUpload -Force
Import-Module -Name AnonUpload -Force               
```

### How to Update:
```powershell
Update-Module -Name AnonUpload -Force      
```
